[3조, 팀장 유환희]

## ▶ 이슈사항

- 백엔드 API 구현 및 프론트엔드 API 연동 보완
  - 피로도 일별 통계 API를 구현하고 프론트엔드 통계 화면과 연결
  - 업체 관리, 차량 관리, 운전자 관리, 운행 이력, 임계값 설정 화면의 API 경로 및 메서드 정합성 점검
  - 로그인 응답의 상태값과 업체 식별값 저장 흐름을 보완하고 승인 대기 계정 화면 이동 처리 확인

- 더미/시드 데이터 보완
  - 테스트용 업체, 관리자, 차량, 운전자, 피로도 임계값 데이터를 정리
  - 데모 계정 로그인 테스트 과정에서 인증 데이터 정합성 보완
  - 실제 기능 테스트에 사용할 기본 데이터 기준을 산출물과 맞춰 정리

- Docker PostgreSQL 실행 및 테스트 과정 보완
  - Docker Compose 기반 PostgreSQL 실행 흐름 점검
  - 백엔드 실행 및 DB 연결 테스트 과정에서 발생한 환경 변수 관련 문제 보완
  - 로컬 테스트와 Docker 기반 실행 기준을 구분하여 확인

- 산출물 최신화
  - 실제 구현 상태를 기준으로 프로젝트개요서, 요구사항정의서, DB 설계서, ERD, 테이블정의서, 프로젝트구조도 수정
  - 산출물 기준정리표와 산출물 현황 분석 문서를 최신화
  - `v0.2.0` 릴리즈 범위와 다음 단계인 `v0.3.0` 통합 테스트 방향 정리
