# logmile_ui_prototype

Vue.js 개발 이전 단계의 HTML 기반 UI 프로토타입

## 개요

`logmile_fe` 개발 착수 전 화면 흐름과 레이아웃을 빠르게 검토하기 위해 작성된 정적 HTML 프로토타입입니다.  
현재는 참고용으로만 보관하며, 실제 서비스는 `logmile_fe`를 사용합니다.

## 파일 목록

| 파일 | 설명 |
|---|---|
| `index.html` | 서비스 메인 소개 페이지 (v1) |
| `index2_ge.html` | 서비스 메인 소개 페이지 (v2, GE 스타일) |
| `page_flow.html` | 전체 화면 흐름 다이어그램 |
| `dashboard_v2.html` | 대시보드 프로토타입 v2 |
| `dashboard_v3.html` | 대시보드 프로토타입 v3 |
| `dash_a.html` | 시나리오 A 기준 대시보드 화면 |
| `dash_b.html` | 시나리오 B 기준 대시보드 화면 |
| `dash_c.html` | 시나리오 C 기준 대시보드 화면 |
| `ge/index.html` | GE 스타일 변형 메인 페이지 |

## 실행 방법

별도 빌드 없이 브라우저에서 HTML 파일을 직접 열어 확인합니다.

```powershell
Start-Process "logmile_ui_prototype\dashboard_v3.html"
```
